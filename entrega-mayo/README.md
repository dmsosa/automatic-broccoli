## Aprendido <a name = "aprendido"></a>

This project built using vanilla javascript and CSS and Dicoding Submission Purpose (Free Share for study)

A table in HTML is built from a few key tags:

Tag	Description
<table>	Defines the main table container
<thead>	Groups the header rows
<tbody>	Contains the table’s data rows
<tr>	Defines a row
<th>	Defines a header cell (bold and centered by default)
<td>	Defines a data cell

El :lang() CSS pseudo-class encuentra elementos de acuerdo con el idioma que se les asigne con el atributo HTML "lang"

La pseudo-clase :only-child CSS representa un elemento que es el 'hijo unico' de otro. tiene el mismo efecto que :first-child:last-child o :nth-child(1):nth-last-child(1)
