<!DOCTYPE >: Se utiliza para especificar la versión de HTML que se esta utilizando en página.

<html>: Define el inicio y el final de la pagina.

<head>: Define la sección de encabezado de la página.

<title>: Define el titulo de la página web que aparece en la pestaña del navegador.

<body>: Define la sección del cuerpo de la página web,donde se incluyen todos los elementos que se mostráran en la pagina.

<h1> a <h6>: Define los titulos de diferentes niveles de jerarquía en la pagina web.

<a>: Define un enlance que el usuario puede hacer click para ir a otra página web o a una seccíon diferente de la misma.

<img>:Define una iamgen que se mostrará enla pagina web.

<p>Etiqeta de párrafo</p>:Define un párrafo de texto.Esta etiqueta formatea cualquier texto entre la etiqueta <p> de apertura y la etiqueta </p> de cierre como párrafo etsándar o texto de cuerpo principal.

<b>Etiqueta de negrita</b>: Esta etiqueta dará formato de negrita a cualquier texto que se encuentre dentro de las etiquetas de cierre y apertura.

<i> Etiqueta de cursiva</i>: Esta etiqueta formateará como cursiva cualquier texto que se encuentre dentro de la etiqueta de apertura t etiqueta de cierre.

<u>Etiqueta de subrayado</u>: Esta etiqueta formateará cualquier texto entre la etiqueta de apertura y cierre con un subrayado.
Ejemplos de una estructura en orden jerarquico
