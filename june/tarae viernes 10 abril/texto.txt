Ejemplo 1: Página básica HTML
<!DOCTYPE html>
<html>
<head>
<title>Mi primera página</title>
</head>
<body>
<h1>Hola mundo</h1>
<p>Este es mi primer sitio web.</p>
</body>
</html>
Objetivo:
• Comprender la estructura básica de una página web
• Visualizar diferencias mínimas entre navegadores
Ejemplo 2: Aplicando estilos con CSS
<!DOCTYPE html>
<html>
<head>
<title>Ejemplo con CSS</title>
<style>
body {
background-color: lightblue;
font-family: Arial;
}
h1 {
color: darkblue;
}
p {
color: black;
}
</style>
</head>
<body>
<h1>Título con estilo</h1>
<p>Este texto tiene formato gracias a CSS.</p>
</body>
</html>
Actividad sugerida:
• Cambiar colores y tipografías
• Visualizar en distintos navegadores

Ejemplo 3: Diferencias de visualización (botón)
<!DOCTYPE html>
<html>
<head>
<title>Botón</title>
<style>
button {
padding: 10px;
border: none;
background-color: green;
color: white;
}
</style>
</head>
<body>
<button>Haz clic</button>
</body>
</html>
Observación docente: El botón puede verse diferente en cada navegador si no se normalizan
estilos.
Ejemplo 4: Uso de márgenes y cajas (box model)
<!DOCTYPE html>
<html>
<head>
<style>
div {
width: 200px;
padding: 20px;
margin: 30px;
border: 2px solid black;
}
</style>
</head>
<body>
<div>Caja de ejemplo</div>
</body>
</html>
Objetivo:
• Entender cómo el espacio cambia según navegador
Ejemplo 5: Prueba en navegador de texto
<!DOCTYPE html>
<html>
<body>
<h1>Ejemplo sin estilos</h1>
<p>Contenido importante</p>
<img src="imagen.jpg" alt="imagen">
</body>
</html>
Análisis:
• En navegador de texto no se verá la imagen
• Se mostrará solo el contenido textual
 Actividad integradora
1. Crear una página con:
o Título
o Párrafo
o Botón
o Una caja con borde
2. Aplicar estilos CSS
3. Probar en:
o Dos navegadores modernos
o (Opcional) un navegador en modo texto
4. Responder:
o ¿Qué diferencias observas?
o ¿Qué elementos cambian?
El objetivo no es solo programar, sino desarrollar criterio técnico para asegurar compatibilidad y
calidad en los sitios web.
